# School of Cities - Urban Data Analytics and Visualization Coursework

This project includes my studies and coursework for the [Urban Data Analytics and Visualization microcredential](https://schoolofcities.utoronto.ca/learning-sofc/urban-data-professional-advancement/urban-data-analytics-and-visualization/) offered by School of Cities. 

This course offers structured learning on content covered in the [Urban Data Analytics, Visualization, & Storytelling online textbook](https://schoolofcities.github.io/urban-data-storytelling/) authored by the School of Cities.
