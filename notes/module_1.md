# Module 1: Introduction to Urban Data Analytics
## Overview
### Learning Goals
- Describe what urban data is and explain why it is used to understand cities
- Identify and categorize common sources, formats, and types of urban data in real-world contexts, including the Canadian census
- Evaluate how choices in data collection, representation, and visualization introduce bias
### Assessment
- Identify a dataset, understand its structure, and consider how it could support analysis or storytelling
- Build the base for a final data story while practicing how to interpret data in context
### Recommended readings
- Introduction to urban data
- Measuring the city: metrics and indicators
- Subjectivity and objectivity in data 
### Key terms and definitions
- Variable: Measurable attribute or characteristic
- Observation / record: Single unit of measurement in a dataset
- Dataset: Structured collection of related observations
- Metric / indicator: Summary measure used to describe or compare phenomena
  - Ex. Population density, average rent, vacancy rate
- Spatial unit: Defined geographic area used for organizing data
  - Ex. Census tract, neighbourhood, municipal boundary
- Open data: Data made publicly available
- Levels of measurement: Categories describing how variables are defined
  - Nominal, ordinal, interval, ratio
- Bias: Systematic favouring or exclusion of certain perspectives, groups, or outcomes in data collection, interpretation, or presentation
- Metadata: Information describing how data was collected, structured, and defined
- Data storytelling: Using data to communicate insights to support interpretation and action
